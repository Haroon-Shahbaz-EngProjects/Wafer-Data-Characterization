Before the application can run, a few libraies are needed to be installed on the computer.
Libraries needed:
Python3: Can be installed from Microsoft store, 
Pip: https://pip.pypa.io/en/stable/installation/
Pandas: pip install pandas (enter in command line)
Numpy: pip install numpy
csv: pip install python-csv
Sympy: pip install sympy


The files you wish to have converted will be placed in the "InputCSV" folder

The end products will be created in the "OutputCSV" folder

When it comes to the propertites of the cartridges (x increment, x spacing, y increment, y spacing)
Please refer to "Grid Layout for User Inputs" for an visual explanation on how these values are found
